## How Run

1) Install the node packages

```bash
npm i
```
2) Build the project 

```bash
npm run build
```

3) Run the Project 

```bash
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## News API

This app is using News API to search the top headlines.
